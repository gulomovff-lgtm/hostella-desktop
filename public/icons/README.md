# Иконки приложения (под стиль логотипа Hostella)

Сюда кладутся сгенерированные иконки. Имя файла = ключ иконки.
Формат: **SVG** (предпочтительно) или PNG 512×512, прозрачный фон.

Компонент `src/components/UI/AppIcon.jsx` сам подхватывает файл
`public/icons/<key>.svg`. Если файла нет — показывает запасной emoji
(ничего не ломается), поэтому набор можно дополнять постепенно.

## Палитра логотипа
- Тёмно-коричневый контур: `#2e2018`
- Бирюза (бренд): `#0f9688`
- Оранжевый: `#e8943a`
- Коралловый: `#e0553a`
- Кремовый фон: `#f5ead6`

### Категории расходов
- rent          (Аренда)
- utilities     (Коммунальные услуги)
- salary        (Зарплата)
- advance       (Аванс)
- groceries     (Продукты)
- taxes         (Налоги)
- registration  (Регистрация)
- internet      (Интернет)
- ads           (Реклама)
- gas           (Газ)
- electricity   (Электричество)
- water         (Вода)
- repair        (Ремонт)
- other         (Другое)

### Статусы / прочее (по желанию)

- cash, card, qr, transfer
